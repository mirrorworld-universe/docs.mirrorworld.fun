# `GetNFTs`

Get info of NFTs by unabriged params.
### Parameters
`collection|string` - The address of the collection to be queried.

`page|int` - The page number, page > 0.

`page_size|int` - The page size, 0 < page size <= 50.

`orderByString|string` - The address of the NFT to be queried.

`filters|List<GetNFTsRequestFilter>` - The filter items.

`action|Action<GetNFTsResponse>` - The callback function of this method is used to receive the results.

### Returns
`response class` - `CommonResponse<GetNFTsResponse>`
