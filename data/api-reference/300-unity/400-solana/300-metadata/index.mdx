# Metadata Methods | Unity API Reference
*If you want to implement your own UI for a marketplace, the Metadata module provides you with the necessary APIs. By using these APIs, you can create [a marketplace similar to Mirror World](https://jump.mirrorworld.fun/), where you can manage and display metadata related to assets, such as NFTs.*

- [`MWSDK.Solana.Metadata.GetNFTInfo`](/api-reference/unity/solana/metadata/GetNFTInfo)
- [`MWSDK.Solana.Metadata.GetNFTs`](/api-reference/unity/solana/metadata/GetNFTs)
