# `GetTransactionsByWallet`

Get transactions of target user.
### Parameters
`walletAddress|string` - Which wallet you want to get from.

`limit|int` -  The limit count of transactions.

`nextBefore|string` - The transaction offset.

`action|Action<SolResGetTransactionByWallet>` - The callback function of this method is used to receive the results.

### Returns
`response class` - `CommonResponse<SolResGetTransactionByWallet>`

### Code Example
```csharp
string walletAddress = "target wallet address";
int limit = 1;
string nextBefore = "next before";
```
