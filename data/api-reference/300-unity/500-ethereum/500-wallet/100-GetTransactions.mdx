# `GetTransactions`

Get transactions of current user.
### Parameters
`limit|int` - The transaction limit.

`action|Action<CommonResponse<EVMResTransactions>>` - The callback function of this method is used to receive the results.

### Returns
`response class` - `CommonResponse<EVMResTransactions>`

### Code Example
```csharp
int limit = 1;

MWSDK.Ethereum.Wallet.GetTransactions(limit, (res) =>
{
    CommonResponse<EVMResTransactions> resObj = res;
    long code = resObj.code;
    string message = resObj.message;
    EVMResTransactions transactionsRes = resObj.data;
    //todo...

});
```
