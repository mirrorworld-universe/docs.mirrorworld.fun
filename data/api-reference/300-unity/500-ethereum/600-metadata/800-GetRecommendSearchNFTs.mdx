# `GetRecommendSearchNFTs`

Get NFTs by server recommend.
### Parameters
`collections|List<string>` - The list of contract address.

`action|Action<CommonResponse<List<EVMResMetadataRecommendNFTs>>>` - The callback function of this method is used to receive the results.

### Returns
`response class` - `CommonResponse<List<EVMResMetadataRecommendNFTs>>`

### Code Example
```csharp
List<string> collections = new List<string>();
collections.Add("collection 1");
```
