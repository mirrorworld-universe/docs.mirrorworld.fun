# `OpenMarket`
Open market page on native.

### Parameters
`url|string` - Pass in the URL of your Marketplace and open it in the default browser. You can either build your own Marketplace website or [use our template to create one](https://mirrorworld.fun/docs/guides/how-to-create-nft-marketplace-storefront).

### Returns
There is no response.

### Code Example
```csharp
string url = "Your marketplace URL";

MWSDK.OpenMarket(url);
```
