# `transferMATIC`

Transfer Matic From Evm Wallet To Another Evm Wallet

```js
import { type EVMTransferTokensPayloadV2 } from "@mirrorworld/web3.js"

const payload: EVMTransferTokensPayloadV2 = {
  amount: 1,
  gasLimit: 100000,
  gasPrice: 1.5,
  to: "0x80Cd58A56643c958dFC29b85329f9FEda93Bdb56",
  nonce: 696969,
}

await mirrorworld.Polygon.Wallet.transferMATIC(payload)
```
