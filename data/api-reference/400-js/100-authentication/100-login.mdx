# `login`

Authenticates a user using a customized user interface. It supports social
authentication with Google, Twitter, Discord and Apple. This method opens a UI
that authenticates the user.

```ts
import { MirrorWorld, Solana } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_SECRET_API_KEY",
  chainConfig: Solana("mainnet-beta"),
})

const { refreshToken, user } = await mirrorworld.login()
```
