# `createMarketplace`

Create A New NFT Marketplace On Solana

```js
import { type ICreateSolanaMarketplacePayloadV2 } from "@mirrorworld/web3.js"

const payload: ICreateSolanaMarketplacePayloadV2 = {
  // Address of trading currency of marketplace. Defaults to SOL: So11111111111111111111111111111111111111112
  treasury_mint: "So11111111111111111111111111111111111111112",
  // Seller fee basis points
  seller_fee_basis_points: 100,
}

await mirrorworld.Solana.Asset.createMarketplace(payload)
```
