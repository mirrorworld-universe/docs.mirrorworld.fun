# `createMarketplace`

Create A New NFT Marketplace On Ethereum

```js
import { type ICreateEVMMarketplacePayloadV2 } from "@mirrorworld/web3.js"

const payload: ICreateEVMMarketplacePayloadV2 = {
  seller_fee_basis_points: 400,
}

await mirrorworld.Ethereum.Asset.createMarketplace(payload)
```
