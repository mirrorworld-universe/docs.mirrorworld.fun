# `updateMarketplace`

Updates A Sn Existing NFT Marketplace On Ethereum

```js
import { type IUpdateEVMMarketplacePayloadV2 } from "@mirrorworld/web3.js"

const payload: IUpdateEVMMarketplacePayloadV2 = {
  seller_fee_basis_points: 400,
}

await mirrorworld.Ethereum.Asset.updateMarketplace(payload)
```
