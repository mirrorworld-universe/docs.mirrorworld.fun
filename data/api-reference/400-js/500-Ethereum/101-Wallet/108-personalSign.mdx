# `personalSign`

Performs Personal Signing Of A String Message

```js
import { type EVMPersonalSignPayloadV2 } from "@mirrorworld/web3.js"

const payload: EVMPersonalSignPayloadV2 = {
  data: "Message to sign",
}
await mirrorworld.Ethereum.Wallet.personalSign(payload)
```
