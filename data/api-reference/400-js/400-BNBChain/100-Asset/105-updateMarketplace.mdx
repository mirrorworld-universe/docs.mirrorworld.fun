# `updateMarketplace`

Updates A Sn Existing NFT Marketplace On BNB Chain

```js
import { type IUpdateEVMMarketplacePayloadV2 } from "@usemirrorworld/web3.js"

const payload: IUpdateEVMMarketplacePayloadV2 = {
  seller_fee_basis_points: 400,
}
await mirrorworld.BNBChain.Asset.updateMarketplace(payload)
```
