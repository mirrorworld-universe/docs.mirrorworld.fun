# `createMarketplace`

Create A New NFT Marketplace On BNB Chain

```js
import { type ICreateEVMMarketplacePayloadV2 } from "@usemirrorworld/web3.js"

const payload: ICreateEVMMarketplacePayloadV2 = {
  seller_fee_basis_points: 400,
}

await mirrorworld.BNBChain.Asset.createMarketplace(payload)
```
