# `transferToken`
Transfer SPL token to a recipient.

```java
Activity returnActivity = this;
String nonce = "nonce ";
String gasPrice = "gas price";
String gasLimit = "gas limit";
String to = "to wallet address";
int decimals = 1;
String contract = "contract";

MirrorWorld.transferToken(returnActivity, nonce, gasPrice, gasLimit, to, decimals, contract, new MirrorCallback() {
    @Override
    public void callback(String result) {
        String r = (result);
        runInUIThread(holder,r);
    }
});
```
